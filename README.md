# ollama-chat

Swing UI Ollama Chat App, build with Mill.

## Requirements
- [Ollama](https://ollama.com/download) with a downloaded model
- Java 25
- [Mill build tool](https://mill-build.org/mill/index.html)

## Features
- chat with Ollama models
- use chat history
- use MCP tools

## Configuration
Via ollama.yaml in the project root directory or user home directory.
